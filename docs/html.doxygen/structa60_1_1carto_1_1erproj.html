Detailed Description

United Nations (UN) Map of the World, 4170, revisions 11-13. This is a CC image file that has been subsequently tweaked to come up with the base .svg file in use here.

Base SVG image: un-world-map-4170r12-meridians-22x17-2013.svg

The current approach is to assume an equirectangular projection, with the standard parallel at the equator (plate carr√©e).
