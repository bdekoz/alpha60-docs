Sequence a directory formatted with half frames such that the frames crawl upwards vertically from the lower half, to the top half, then off screen. As each image moves up, the next image in series is taken from the input directory until none remains.

idir is a directory of png images, each of which is half the size of a frame

Common resolutions refreshers: 1080p is 1920 x 1080 4k is 3840 x 2160 (aka 2 x 2 of 1080p)

So area = 4k implies an input directory of 2 x (3840 x 1080p) frames stacked vertically.

This will produce as output numbered frames for as many image combinations as is required to use up all the images in the input directory.

Then use a script to transform each image to a movie of n seconds, merge.
